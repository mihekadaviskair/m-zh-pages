# m-zh-pages

一个用于归档和发布独立 HTML 页面的仓库。本仓库不绑定特定域名或网站，主要用于存放各类可直接在浏览器中运行的 HTML 文件。

## 项目简介

本项目旨在提供一个简洁、稳定的 HTML 页面托管空间。所有页面均为独立文件，不含后端依赖，适合用于快速展示静态内容、原型演示、资料归档等场景。

## 目录说明

```
m-zh-pages/
├── pages/        # 存放所有 HTML 页面文件
├── assets/       # 页面共用的静态资源（如图片、样式、脚本）
└── README.md     # 本文件
```

- **pages/**：每个 HTML 文件独立存放，文件名应反映页面内容或用途。
- **assets/**：页面可能引用的公共资源，按需组织。

## 页面归档说明

- 所有页面均为纯前端 HTML 文件，无需服务器环境即可运行。
- 页面内容应当自包含，避免引用外部不可控资源。
- 归档时请确保页面可正常渲染，无功能性错误。
- 本仓库不限制页面主题或用途，但需符合一般内容规范。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面。
- 请为每个页面提供简洁的注释或说明（可写在 HTML 注释中）。
- 不建议在页面中嵌入恶意代码、跟踪器或商业推广内容。
- 本仓库由维护者定期审查，可能移除不符合规范或长期未更新的页面。

## 使用方式

1. 将 HTML 文件放入 `pages/` 目录。
2. 如有需要，将相关资源放入 `assets/` 目录，并在页面中正确引用。
3. 提交 Pull Request 或直接推送（视权限而定）。

## 许可

本仓库下的 HTML 文件默认采用 MIT 许可，除非文件内另有声明。详情请参阅仓库根目录下的 LICENSE 文件（如存在）。
